# 🛠️ Instructions

## 🔹 How to Change the Video

1. Open `src/App.jsx`
2. Replace the video URL:
   ```jsx
   <ReactPlayer url="https://vimeo.com/YOUR_VIDEO_ID" controls />
   ```
3. Save the file and refresh the page.
